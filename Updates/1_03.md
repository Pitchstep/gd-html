# Update 1.03 - Update Log

You're looking at it.

## What changed
- Added a version number to the main menu.
- Added this Update Log screen, which reads changelog entries straight from Markdown files in the `Updates/` folder.
- Each entry gets a blurred background thumbnail, a title, and a date in the sidebar — click one to read the full notes.

## Why it matters
Every future change now gets recorded somewhere players can actually see it, instead of living only in a chat conversation. To add a new entry later, just drop a new `Updates/N.md` file and add one line to `manifest.json` — no code changes needed! :D (This is if you want to host your own version of the game!)
