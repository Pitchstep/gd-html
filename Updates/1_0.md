# Update 1.0 - Editor Updates

The level editor is now fully functional! And we have a Version Number!!!!

## What changed
- Fixed a bug where the editor screen silently absorbed all clicks, so nothing you placed ever actually landed on the grid.
- Added click-and-drag painting — hold and drag across the grid to place a run of tiles instead of clicking one at a time.
- Added right-click to erase a tile instantly, no tool-switching required.
- Added a live hover outline so you can see exactly which cell you're about to affect before you click.
- Added a Version Number to the main menu!

## Why it matters
Building anything more than a couple of blocks used to mean a lot of repetitive clicking. This should make blocking out longer levels much faster!
